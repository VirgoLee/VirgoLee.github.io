1. 概述

最开始是通过 Redis 官网下载的文件安装的，这里使用 Docker 在安装部署一次。

主要分为3个步骤：

  • 1.拉取 Redis 镜像
  • 2.准备配置文件
  • 3.启动容器

当然 需要提前安装 Docker。

2. 拉取 Redis 镜像
